# bundlewrap-hostname

`bundlewrap-hostname` is a small and simple bundle. It only exists to set the hostname to the node name by using `hostnamectl`.

## Compatibility

This bundle has been tested on the following systems:

| OS          | `[x]` |
| ----------- | ----- |
| Fedora 24   | `[x]` |
| Fedberry 24 | `[x]` |
